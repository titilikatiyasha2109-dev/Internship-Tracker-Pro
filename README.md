
# Internship Tracker

A simple web application built to track internship applications and related details.

## Tech Stack
- Vite
- React
- TypeScript
- HTML
- CSS

## Project Purpose
This project is created for learning and practice purposes.  
It focuses on understanding project structure, components, and basic services in a React + TypeScript setup.

## Folder Structure
- `components/` – UI components
- `services/` – Service and helper logic
- `App.tsx` – Main application component
- `index.tsx` – Application entry point
